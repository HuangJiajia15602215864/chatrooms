# chatrooms
利用node实现事件驱动的聊天室程序，客户端实时采用webSocket方案。


## 代码目录

```
|-- lib              服务端逻辑
     * chat_server.js
|-- node_modules     项目依赖包（不上传到GitHub）
|-- public           静态文件
     * index.html    HTML页面
     * stylesheets   css文件
     * javascripts   js文件
|-- .gitignore        忽略的文件
|-- package.json      项目描述文件
|-- README.md         文字介绍项目详情
